# `url-parser`

Test it [`here`](https://onlinegdb.com/REo65_RxH)

<br>

## Requirements

- `make`
- `cmake`

---

<br>

## Commands

### Building

```console
make
```

```console
cd build/
```

```console
cmake ..
```

```console
cmake --build .
```

### Executing

#### Parse URL

```console
./build/app/app https://mysite.com/video?time=123s
```

```console
./build/app/app https://mysite.com/?redirect=%2fcontact%2femail --decode
```

#### Demo

```console
./build/app/app --demo
```
